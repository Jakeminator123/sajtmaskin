## Recommended Actions

**A. Baseline**
```bash
cd /Users/christophergenberg/Desktop/Sajtmaskin_Genberg
git status
git branch --show-current
git fetch origin
git rev-parse HEAD
git rev-parse origin/master
git merge-base HEAD origin/master
git rev-list --left-right --count HEAD...origin/master
```

**B. commit_first (avsiktlig WIP)**
```bash
git add <scoped paths>
git commit -m "wip(frontend): scoped WIP before origin/master merge"
git merge origin/master
# resolve → git add → git commit
```

**C. stash**
```bash
git stash push -u -m "pre-merge-frontend-christopher-$(date +%Y%m%d)"
git merge origin/master
# efter typecheck+lint+vitest (nedan):
git stash pop
```

**D. Efter merge**
```bash
npm run typecheck && npm run lint && npx vitest run
# regex? ev. node scripts/dev/check-unicode-regex.mjs
```

**E. Push (frivilligt)**
`git push origin frontend/christopher` utan `--force`.
