# Agent Reports

Denna katalog ar kanonisk plats for agentrapporter i parallella pass.

## Struktur
- En rapport per pass: `docs/agent-reports/<agent-pass-namn>.md`
- Gemensam mall: `docs/agent-reports/_template.md`

## Minimikrav
- Nulage (vad som ar sant idag)
- Problem/risker med sannolikhet + paverkan
- Berorda filer
- Rekommenderat nasta smala steg
- Tydligt vad som ar gjort och vad som inte ar gjort

## Namngivning
- Hall filnamn korta och beskrivande, t.ex.:
  - `agent-01-compat-v0-audit.md`
  - `agent-02-preview-lifecycle.md`
  - `agent-07-followup-routeplan-eskalering.md`
