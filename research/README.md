# Research Lane

Non-runtime research inputs and curated reference material.

Runtime code never imports from this directory. Instead, build scripts
under `config/scripts/` read the normalized catalog here and produce
generated artifacts committed under `src/lib/gen/`.

## Directory structure

```
research/
  normalized-catalog.json   Normalized template catalog (from Zone 1 scrape)
  README.md                 This file
```

## Normalized catalog

`normalized-catalog.json` is the bridge between raw scrape data (Zone 1,
outside the repo) and runtime artifacts (Zone 3). It is generated by:

```bash
npm run research:normalize -- --input ../vercel-scrape
```

## Build pipeline (after catalog update)

```bash
npm run template-library:rebuild    # template-library JSON + embeddings
npm run scaffolds:build             # research stubs + scaffold embeddings + validate
npm run verify:generated-paths      # ensure no machine-specific paths
```
