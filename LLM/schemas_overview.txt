SAJTMASKIN SCHEMAS - OVERVIEW
Purpose: Samla en standard for hur anrop och floden fungerar.
Scope: Endast beskrivande dokument (ingen runtime).

Files in this folder:
- calls.txt
  Standard for viktigaste API-anrop, skillnad gateway/inte gateway,
  och vilka filer som ansvarar for vad.

Relation to other docs:
- src/...: kodkalla for sanning, schemas maste uppdateras nar kod andras.

How to update:
0) Uppdatera endast pa uttrycklig begaran.
1) Nar ett endpoint, request eller response andras -> uppdatera calls.txt.
2) Nar nya floden tillkommer -> lagg till under "Flows" i calls.txt.
3) Nar naming eller konfiguration andras -> uppdatera "Env/Config" i calls.txt.
4) Nar v0-systemprompt eller prompt-assist beteende andras -> uppdatera calls.txt.
5) Nar Custom Instructions-flode andras -> uppdatera calls.txt + builder-noter.
6) Nar post-generation checks utokas (CSS/unicode) -> uppdatera post-generation-checks.txt.

Terminologi:
- Platform API (v0-sdk) = kodgenerering + preview.
  Canonical tiers: v0-max-fast (default), v0-1.5-lg, v0-1.5-md, v0-gpt-5.
  Legacy aliases: v0-mini -> v0-1.5-md, v0-pro -> v0-1.5-md, v0-max -> v0-max-fast.
  Source of truth: src/lib/v0/models.ts
- Model API (AI SDK / @ai-sdk/openai) = textmodell (v0-1.5-md/lg).
  Anvands enbart for prompt assist, ej for sajtbygge.

Designprincip:
- Kort och exakt. Peka till kod, duplicera inte logik.
- Inga hemligheter eller nyckelvarden i texten, men kan skrivas i chatten och användas fritt där och i "env"-filer.
- Teknisk standard och sprakregel ligger i DEFAULT_CUSTOM_INSTRUCTIONS (src/lib/builder/defaults.ts).

Wizard enrich (2026-03):
- Endpoint: POST /api/wizard/enrich
- Request:
  - mode: "step" | "final_check" (default "step")
  - step: 1..5
  - data:
    - companyName, industry, location, existingWebsite
    - inspirationSites[]
    - purposes[], targetAudience, usp, selectedVibe, specialWishes
    - previousFollowUps: Record<string,string>
  - scrapeUrl?: string
- Response:
  - questions[]: { id, text, type, options?, placeholder?, priority?, dependsOn? }
  - suggestions[]: { type, text }
  - insightSummary?: string | null
  - meta?: { confidence?, needsClarification?, unknowns?, priority? }
  - scrapedData?: { title?, description?, headings?, wordCount?, hasImages?, textSummary? }
  - contextHash?: string
- Not:
  - mode=final_check anvands for AI-klargoranden innan slutbrief i wizardn.
  - previousFollowUps ska alltid beaktas for att undvika repetitiva fragor.
