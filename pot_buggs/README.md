# pot_buggs

Arbetsyta för potentiella buggar och granskningar.

- **`done/`** — korta, avslutade rapporter (vad som fixats, datum i filnamn).
- **`buggar _llm/`** — temporära utkast (ignoreras av git; kan raderas lokalt när som helst).

Körningar som landar i produktionskod dokumenteras även i `docs/architecture/` när det gäller flöden.
