Legender — nyckelfiler

Prompt-förbättring

promptAssist.ts — system prompts för polish/rewrite
usePromptAssist.ts — klientlogik + guardrails
/api/ai/chat — shallow polish/rewrite endpoint
/api/ai/brief — deep brief endpoint

Orkestrering

promptOrchestration.ts — deterministisk budget/kompressionsvakt
orchestrate.ts — scaffold + route plan + kontrakt → BuildSpec
system-prompt.ts — bygger statisk + dynamisk systemprompt

Generation

engine.ts — huvud-LLM via streamText
phase-routing.ts — modellval per fas (manifest-styrd)
build-spec.ts — styrdata för generering

Reparation

pipeline.ts — deterministisk autofix (loop)
validate-and-fix.ts — esbuild syntax + LLM fixer
llm-fixer.ts — LLM-driven kodreparation
server-verify.ts — bakgrundsverifiering + repair loop
repair/route.ts — manuell repair-endpoint

Konfiguration

manifest.json — central LLM-konfiguration
load-manifest.ts — Zod-validering + runtime getters
defaults.ts — runtime-konstanter (manifest + env)
app.py — Streamlit dashboard/GUI
