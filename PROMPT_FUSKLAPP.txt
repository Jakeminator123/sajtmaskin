PROMPT FUSKLAPP - promptAssist / defaults
=========================================

Vad som andrades (kort)
- promptAssist.ts: gemensam helper for shadcn-setup och delad guidance
  (motion, visual identity, quality bar, image density) for att ta bort dubbletter.
- defaults.ts: kortade duplicerade rader for att minska token-bloat.
- Inga nya krav lades till; textinnehall ar samma eller kortare.

Varfor (tanke)
- Mindre duplication ger samma budskap med farre tokens och enklare underhall.
- Samma instruktioner ateranvands konsekvent mellan promptvarianter.

Hur du far tillbaka tidigare promptlangd/logik
Alternativ A (enklast om git finns):
1) git restore "src/lib/builder/promptAssist.ts" "src/lib/builder/defaults.ts"

Alternativ B (manuellt):
1) I promptAssist.ts, aterinfor SHADCN_SETUP_BLOCK och de separata listorna
   for motion/visual/quality/image.
2) I buildV0PromptFromBrief, ersatt getShadcnSetupLines("detailed")
   med SHADCN_SETUP_BLOCK.
3) I buildDynamicInstructionAddendumFromBrief och buildDynamicInstructionAddendumFromPrompt,
   ersatt konstanterna med de hardkodade listorna.
4) I defaults.ts, aterstall originalraderna i "Tech Stack", "Component Usage"
   och "Tailwind Best Practices".

Snabb kontroll
- Jamfor diff i git eller anvand IDE-historik for att hitta exakt tidigare text.
