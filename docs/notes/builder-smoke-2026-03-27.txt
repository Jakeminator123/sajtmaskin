Rotorsak: "React is not defined"
---------------------------------
- Ändringen i commit b00f6e4a7 (eller närliggande storstädning) bytte
  React UMD CDN i src/lib/gen/preview/index.ts från @18 till @19.
- React 19 publicerar INTE UMD-bundles (unpkg ger 404 på /umd/react.production.min.js).
- Preview-skriptet kör tusentals rader som beror på globala React/ReactDOM (window.React).
  Utan UMD → "React is not defined" vid körning.
- Fix: byt tillbaka till react@18 + react-dom@18 UMD i preview/index.ts.
  React 18 UMD sätter window.React/window.ReactDOM och är API-kompatibelt
  med den genererade JSX-koden.

Fix 2: Reasoning/thinking aktiveras nu mot OpenAI API
------------------------------------------------------
- engine.ts hade thinking=true som default men skickade aldrig
  providerOptions.openai.reasoningEffort till streamText().
- GPT-5.4 default reasoning effort = "none" → noll reasoning utan explicit begäran.
- Fix: engine.ts skickar nu { openai: { reasoningEffort: "high" } } via providerOptions
  när thinking=true och modellen är OpenAI (ej Anthropic).
- models.ts: ny export isAnthropicModel() för att skilja OpenAI/Claude-banorna.
- Gäller alla OpenAI GPT-5.x-modeller (gpt-5.4, gpt-5.3-codex, gpt-5.1-codex-max, gpt-5.2).
- Anthropic Claude har egen extended-thinking-mekanism via @ai-sdk/anthropic som
  behöver separat arbete (anteckning, inte fixat nu).
