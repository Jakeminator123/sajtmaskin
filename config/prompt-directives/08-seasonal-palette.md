# Seasonal & Thematic Palette
<!-- directive: seasonal-palette -->
<!-- cascade: explicit > indicated > inferred > default -->

When the prompt or brief indicates a seasonal or cultural theme, override the default palette with subject-appropriate colors. This directive is a fallback — if the brief's `colorPalette` already specifies the right colors, use those directly.

## Christmas / Winter / Holiday
<!-- keywords: jul, christmas, holiday, festive, vinter, winter, gran, granar, tree, julmarknad, snow, snö -->

- Use a subject-led palette instead of default SaaS blue.
- Suggested subject palette: evergreen/spruce green, deep Christmas red, snow white, bark brown, and warm gold.
- Use those colors in hero backgrounds, CTA accents, badges, borders, and decorative details instead of default SaaS blue.

## Autumn / Harvest
<!-- keywords: höst, autumn, harvest, thanksgiving, halloween -->

- Warm earth tones: burnt orange, deep amber, russet brown, olive green.
- Use muted, warm backgrounds with rich accent colors.

## Spring / Fresh
<!-- keywords: vår, spring, bloom, påsk, easter -->

- Light pastels: soft pink, mint green, lavender, butter yellow.
- Airy backgrounds with gentle gradients and botanical accents.

## Summer / Tropical
<!-- keywords: sommar, summer, tropical, beach, strand -->

- Vibrant warm tones: coral, turquoise, sunset orange, palm green.
- High-energy contrast with warm backgrounds.

## Dark / Gothic / Noir
<!-- keywords: dark, gothic, noir, mörk, svart, black, midnight -->

- Deep blacks and charcoals with jewel-tone accents: ruby, emerald, sapphire, gold.
- Use noise textures, vignettes, and dramatic lighting contrasts.

## General Rule

- Use a subject-led palette instead of default SaaS blue. Seasonal/cultural themes should borrow color cues from the actual subject matter.
