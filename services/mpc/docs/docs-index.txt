Sajtmaskin Documentation Index
==============================

Single källa: dokumentation finns i `services/mpc/docs` och `LLM/` och exponeras som
MCP-resources via `docs://local/...`.

Så läser du
-----------
- MCP-servern exponerar endast resources (inga verktyg).
- Top-level-filer i `docs/` exponeras som `docs://local/{name}`.
- LLM-filer exponeras som `docs://local/llm/{name}`.
  Exempel: `docs://local/llm/v0-prompt-guide`
Obs: Vissa äldre .md-filer hänvisar till tools som inte är aktiverade i servern.

Prioritet
---------
1. WHEN-TO-USE-MCP.md — När och hur du ska använda MCP-servern (LÄS FÖRST!)
2. quick-reference.txt — snabböversikt och vanliga mönster
3. overview.txt — server/struktur
4. error-playbook.txt — felrapportering/loggning
5. LLM/*.txt — aktuella AI-/v0-notes

Lägga till eller flytta docs
----------------------------
- Lägg nya filer som .txt/.md/.json under `services/mpc/docs/local/`.
- Starta om MPC-servern om den körs (`npm run mpc`) så indexet uppdateras.

Scraped docs
------------
- Om `docgrab__*` finns under `docs/` exponeras de också som resources.

Tips
----
- Håll allt i `services/mpc/docs` och `LLM/`.
- Lägg inte hemligheter i docs/.
