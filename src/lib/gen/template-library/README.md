# Template Library (cursorignored generated data)

Large JSON is cursorignored. Context: [`../README.md`](../README.md), [`../../../../docs/architecture/repository-and-platform.md`](../../../../docs/architecture/repository-and-platform.md) (Vercel mall / generated artifacts).

## Files

| File | Size | What it is |
|------|------|-----------|
| `template-library.generated.json` | ~6 MB | 64 curated template entries with metadata, quality scores, strengths, stack tags, and `selectedFiles` (code excerpts from dossiers). Runtime derives structured guidance from this metadata (`style rules`, `section inventory`, `avoid patterns`, `world-class rubric`) before falling back to snippets. Generated by `scripts/template-library/build-template-library.ts` from `research/external-templates/`. |
| `template-library-embeddings.json` | ~3 MB | OpenAI `text-embedding-3-small` vectors for each template entry. Used by `search.ts` for semantic search at runtime. |
| `catalog.ts` | small | Loads and caches entries from the generated JSON. |
| `search.ts` | small | `searchTemplateLibrary(query)` — keyword + embedding search when `entries[]` is non-empty. If the curated catalog is **empty**, returns `[]` immediately (stale `template-library-embeddings.json` is **not** loaded and does not call OpenAI). |
| `types.ts` | small | TypeScript types for entries, search results, selected files. |
| `embeddings-core.ts` | small | Embedding generation utilities. |
| `index.ts` | small | Barrel export. |

## Runtime flow

1. `system-prompt.ts` calls `rankTemplateReferences(prompt, scaffold)` which calls `searchTemplateLibrary(prompt)`
2. With an empty catalog, step 1 yields no prompt-driven matches; `scaffold-research.generated.json` `referenceTemplates` also resolve only if those IDs exist in the catalog.
3. Top matches are injected as structured "Relevant Template References" in the dynamic system prompt with rule-oriented guidance (style rules, sections, avoid list, rubric) before any code excerpts.
4. A smaller subset also gets `selectedFiles` (code excerpts) injected as "Reference Code Snippets" when prompt budget allows.
5. Runtime/test validation treats unknown scaffold families or stale scaffold `referenceTemplates` ids as failures rather than silent fallback.

## Regeneration

Run `scripts/template-library/build-template-library.ts` to rebuild from dossiers in `research/external-templates/reference-library/`.
Run `scripts/embeddings/generate-template-library-embeddings.ts` (or `npm run template-library:embeddings`) after the catalog has `entries[]` again.
