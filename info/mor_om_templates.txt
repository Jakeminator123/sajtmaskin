How to Use These Templates

Find the template ID: From a template’s URL on v0.app, copy the part after the last hyphen. This is the v0TemplateId used by the API. For example, the URL https://v0.app/templates/modern-agency-website-liquid-glass-ezmvVsZJxz8 has the ID ezmvVsZJxz8
v0.app
.

Initialize the template via API: Use the v0 Platform API to create a new chat from the template by calling v0.chats.init with { templateId: "XYZ123..." }. The API will return the template’s files and a live demoUrl for preview.

Customize via refinement: After initialization, you can send refinement prompts through v0.chats.sendMessage to modify the template (e.g., changing colours, adding sections). This uses the platform API’s conversation workflow【11†L101-L110】【11†L130-L139】.

Add more templates: If you want more options, browse the v0 templates gallery (Landing Pages, Dashboards, Websites) to find additional templates and note their IDs.

Tip: When building a site builder, keep a registry (local-templates.ts) of template names, categories, and IDs so users can easily select from curated options. Provide a preview image and description for each entry to improve user experience.
