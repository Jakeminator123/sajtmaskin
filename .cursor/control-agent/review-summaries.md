# Control Agent Review Summaries

Det här är den lilla synliga indexytan för `/control-agent`.

Använd formatet:

## <YYYY-MM-DD>-<slug> (archived <YYYY-MM-DD> <HH:MM>)
- **Scope:** Kort rad om vad som granskades.
- **Status:** `CLEAR` eller `ISSUES FOUND`.
- **Top findings:** 1-3 korta punkter eller `None`.
- **Archive path:** `.cursor/control-agent/archive/<YYYY-MM-DD>-<slug>-<HHMMSS>/`

Håll varje entry kort. Full lokal review-kontext hör hemma i arkivet, inte här.
