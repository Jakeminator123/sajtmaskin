# Handoffs (arkiverat)

Tidigare låg tidsbundna arbetsström-briefar som `*.md` här. **De är borttagna** från nuvarande träd så att kanoniska docs inte duplicerar historik.

- **Fulltext:** `git log` och historiska commits under sökvägen `docs/handoffs/` (innan borttagning).
- **Operativ backlog och beslut:** [`../plans/active/PROJECT-STATE-AND-DIRECTION.md`](../plans/active/PROJECT-STATE-AND-DIRECTION.md)
- **Arkitektur / preview / generation:** [`../architecture/README.md`](../architecture/README.md), särskilt [`preview-deploy.md`](../architecture/preview-deploy.md) och [`builder-generation.md`](../architecture/builder-generation.md).

Lägg inte nya långa handoff-dokument här — uppdatera kanoniska filer ovan eller `PROJECT-STATE-AND-DIRECTION.md`.
