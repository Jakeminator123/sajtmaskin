Sajtmaskin Documentation Index
==============================

Project-specific docs exposed via the `sajtmaskin-mpc` MCP server.
External docs (v0, shadcn, Vercel, OpenAI) are handled by dedicated MCP servers.

How it works
------------
- The server exposes **resources only** (no tools).
- Files in `docs/` -> `docs://local/{name}`
- Files in `LLM/` -> `docs://local/llm/{name}`

Files in this server
--------------------
- WHEN-TO-USE-MCP.md - Which MCP server to use for what
- quick-reference.txt - API patterns, models, code examples
- overview.txt - Server structure
- docs-index.txt - This file

LLM docs (docs://local/llm/...)
--------------------------------
- llm/AC-schema
- llm/DEPS-STATUS
- llm/INDEX
- llm/atgardsmatris
- llm/mijo-logik
- llm/schemas_overview
- llm/v0-prompt-guide

Adding docs
-----------
- Add .txt/.md/.json files under `services/mpc/docs/` or `LLM/`.
- Restart the MPC server (`npm run mpc`) to re-index.
