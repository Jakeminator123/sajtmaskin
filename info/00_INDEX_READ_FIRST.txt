================================================================================
                    PROJECT SUMMARY
================================================================================

WHAT WE'RE BUILDING:
A platform where users can create their own websites using AI.
Users interact with OUR frontend, but v0's API powers the generation.
v0 should be completely invisible to end users (white-labeled).

TECH STACK:
- Next.js 14+ (App Router)
- React 18+
- TypeScript
- Tailwind CSS
- ShadCN UI
- v0 Platform API (hidden behind our backend)

KEY FEATURES:
1. Template selection (12 categories, properly segmented)
2. Free-form prompt input
3. Live preview (self-hosted, not v0's iframe)
4. Iterative refinement via chat
5. Code export (ZIP, GitHub)
6. One-click deploy (optional)
