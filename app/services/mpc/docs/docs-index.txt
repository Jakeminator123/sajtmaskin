MCP TOOLS AVAILABLE
-------------------

search_docs(query, source, limit)
  - Search all documentation for a term
  - source: "all" | "ai-sdk" | "openai" | "vercel" | "v0" | "local"
  - Example: search_docs("streamText", "ai-sdk")

get_doc(docPath)
  - Read a specific documentation file
  - Example: get_doc("docgrab__ai-sdk.dev__docs/llms/llms.txt")

list_doc_sources()
  - List all available sources and file counts

report_error(message, level, stack, component, context)
  - Log an error for later inspection

list_errors(limit)
  - Get recent error log entries

See docs/information.md for detailed project documentation.
