Sajtmaskin MCP Server
=====================

Project-specific documentation server for AI agents.
External docs are handled by dedicated MCP servers (v0, Vercel, shadcn, OpenAI).

Entry point: services/mpc/server.mjs (Node, ESM, stdio transport)
Start: npm run mpc

WHAT IT DOES
------------
Exposes resources only (no tools) via docs://local/{name}.
Files in services/mpc/docs/ and LLM/ are indexed automatically.

RESOURCES
---------
- WHEN-TO-USE-MCP.md  : Which MCP server to use for what
- quick-reference.txt : API patterns, models, code examples
- overview.txt        : This file
- docs-index.txt      : Documentation map
- LLM/*.txt           : AI/prompt strategy notes

LOG ROTATION
------------
Access log auto-rotates at 500 entries.
Location: services/mpc/logs/access-log.jsonl
