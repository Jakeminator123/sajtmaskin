Det finns flera bra sätt att importera och använda V0-templates i din applikation:

1. Shadcn CLI (Rekommenderat)
Det enklaste sättet är att använda shadcn CLI för att installera komponenter direkt:

npx shadcn@latest add "https://v0.dev/chat/b/[BLOCK_ID]"
Detta installerar komponenten och alla dess beroenden automatiskt i ditt projekt.

2. Download ZIP
I V0-gränssnittet kan du klicka på de tre prickarna i övre högra hörnet av din block-vy och välja "Download ZIP". Detta ger dig hela projektet som du kan extrahera och använda.

3. GitHub Integration
Du kan koppla din V0-chat till ett GitHub-repo via sidebar-menyn (Git-ikonen). Alla ändringar pushas automatiskt till den valda branchen, vilket gör det enkelt att synka med ditt huvudprojekt.

4. Kopiera komponenter manuellt
Du kan kopiera enskilda komponenter från V0 till ditt projekt. Se till att:

Kopiera alla nödvändiga filer (komponenter, styles, assets)
Installera beroenden som anges i package.json
Uppdatera import-paths efter din projektstruktur
Rekommendation
Shadcn CLI är det smidigaste alternativet eftersom det hanterar beroenden och konfiguration automatiskt. Om du bygger ett större projekt med flera templates, använd GitHub-integrationen för att hålla allt synkroniserat.

interface Template {

import { openai } from '@ai-sdk/openai'
